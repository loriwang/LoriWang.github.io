Vue中引入阿里图标

发表于 2017-12-14 | 分类于 vue

最近在学习vue,涉及到一个父子组件通信,觉得有必要来写下来了解一下

一.在iconfont中建立项目

iconfont 地址

二.搜索图标并加入到购物车

三.购物车图标添加到项目中

打开右上角购物车面板、选择并添加到项目中

四.选择引入方式

打开项目面板选择你喜欢用字体的形式引入

五.生成链接

第一次使用点击生成链接、之后添加图标再更新链接就可以了

六.引入

新建或者在原有的css文件中加入一句@import url (iconfont生成的链接)
